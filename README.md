# 📝 معرفی پروژه

این پروژه یک سیستم هوشمند مبتنی بر **بینایی ماشین (Computer Vision)** است که با تحلیل موقعیت و ابعاد چهره در فریم‌های ویدیویی، فاصله کاربر تا دوربین را تخمین زده و به‌صورت خودکار عملیات **زوم دیجیتال هوشمند (Digital Auto-Zoom)** را روی چهره انجام می‌دهد.

سیستم ابتدا با استفاده از الگوریتم **Haar Cascade** چهره را شناسایی می‌کند و سپس بر اساس اندازه کادر چهره (`w` و `h`) میزان زوم مناسب را تعیین می‌کند تا خروجی نهایی نرم، دقیق و با کمترین افت کیفیت باشد.

# ⚡ ویژگی‌های فنی پروژه

## 🎥 پردازش تصویر HD

تنظیم مستقیم وب‌کم روی رزولوشن:

```text
1280×720 (720p HD)
```

جهت دریافت جزئیات بیشتر برای زوم بهتر و تشخیص دقیق‌تر چهره.

## 🧠 پردازش اولیه تصویر

- تبدیل فریم‌ها به حالت خاکستری (GrayScale)
- حذف نویز محیط با استفاده از Gaussian Blur
- افزایش دقت تشخیص چهره در نور و شرایط مختلف

## 🔍 سیستم زوم هوشمند سه‌مرحله‌ای

| وضعیت چهره | اندازه چهره | ضریب زوم |
|---|---|---|
| نزدیک به دوربین | بیشتر از 300px | `2.0X` |

| دور از دوربین | کمتر از 150px | `1.2X` |
