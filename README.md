# 📝 درباره پروژه

این پروژه یک سیستم هوشمند در حوزه **بینایی ماشین (Computer Vision)** است که با تحلیل ابعاد چهره کاربر در فریم‌های ویدیویی، فاصله او را تا دوربین تخمین زده و به صورت خودکار و کاملاً داینامیک، عملیات زوم دیجیتال (**Digital Auto-Zoom**) را روی چهره پیاده‌سازی می‌کند.

برنامه با استفاده از الگوریتم لایه‌ای **Haar Cascade** ابتدا موقعیت دقیق چهره را شناسایی کرده و سپس با ایجاد یک سیستم منطق فازی بر اساس عرض و ارتفاع کادر (`w` و `h`)، میزان بزرگ‌نمایی را بهینه‌سازی می‌کند تا خروجی تصویر فوکوس‌شده، نرم و بدون افت کیفیت شدید باشد.

# ⚡ ویژگی‌های فنی و معماری الگوریتم

- **رزولوشن بالا و پردازش HD:**  
  پیکربندی مستقیم وب‌کم روی رزولوشن **1280×720 (720p HD)** جهت دریافت بیشترین جزئیات ممکن از چهره برای زوم‌های باکیفیت‌تر.

- **بهینه‌سازی تصاویر با فیلتر گوسی (Gaussian Blur):**  
  تبدیل فریم‌ها به حالت خاکستری (**GrayScale**) و اعمال فیلتر گوسی مات‌کننده برای حذف نویزهای فرکانس بالا و پیکسل‌های مخرب محیطی قبل از عملیات پردازش چهره.

- **سیستم زوم ۳ مرحله‌ای هوشمند (Dynamic Zoom Factor):**

  - **چهره نزدیک (بزرگ‌تر از ۳۰۰ پیکسل):** زوم عمیق با ضریب `2.0`
  - **چهره متوسط (بین ۱۵۰ تا ۳۰۰ پیکسل):** زوم متعادل با ضریب `1.5`
  - **چهره دور (کوچک‌تر از ۱۵۰ پیکسل):** زوم نرم با ضریب `1.2` به همراه بازسازی ابعاد تصویر.

- **دریافت تصاویر نرم با Interpolation:**  
  استفاده از الگوریتم درهم‌سازی مکعبی تصویر (`cv2.INTER_CUBIC`) برای نرم کردن لبه پیکسل‌ها هنگام بزرگ‌نمایی، و الگوریتم (`cv2.INTER_AREA`) برای جلوگیری از دندانه‌دار شدن تصاویر دور.
